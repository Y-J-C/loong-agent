# raw 原始证据说明

raw 目录用于保存第一、第二、第三阶段的原始只读采集证据。所有 raw 文件按阶段归档，不改写原始内容。

| 目录 | 内容 | 说明 |
|---|---|---|
| `raw/stage1/` | 第一阶段环境归档 | 包含初始环境、dmesg、apt policy、pip 状态、systemd failed 详情等 |
| `raw/stage2/` | 第二阶段硬件/系统/外设只读采集 | 不含修复操作，不含写盘操作 |
| `raw/stage3/` | 第三阶段软件栈/包管理证据 | 包含 apt policy、dpkg、runtime 检查等合并证据 |

## 使用原则

1. raw 文件只用于追溯证据；
2. 不根据 raw 文件直接执行修复命令；
3. 不修改 raw 文件内容；
4. 若发现文档结论与 raw 证据冲突，以 raw 证据为优先，并在后续 changelog 中修正。
